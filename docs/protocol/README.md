# Protocol

These documents define Mokabook's implementation contract. Package, authoring,
static build/check, responsive Browse, watched development, Review, packed
consumer verification, CI, and npm release automation describe implemented
pre-release behavior. The first public release and downstream Accounting
cutover remain external delivery steps.
